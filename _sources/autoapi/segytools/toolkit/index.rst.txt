.. py:function:: textual_header(input_segy_file, txt_encoding) -> str

   :param input_segy_file: Input segy file path from which to extract the textual header.

   :param txt_encoding: Must be either 'ebcdic' or 'ascii'

   :raises ValueError:

   :rtype: str
